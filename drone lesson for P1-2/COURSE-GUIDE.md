# 🚁 無人機編程課程（P1-P2）

## 形式
- 100% HTML 互動課程，積木式編程（類似 Scratch）
- 無需實體無人機，瀏覽器即可運行

---

## 🧩 積木式編程（6堂）

| 堂數 | 積木技能 | 任務 |
|------|---------|------|
| 第1課 | 起飛 🚀 | 讓無人機從地面起飛 |
| 第2課 | 前 / 後 / 左 / 右 ⬆️⬇️⬅️➡️ | 讓無人機走一個正方形 |
| 第3課 | 重複 🔄 | 用最少的積木讓無人機巡邏直線 |
| 第4課 | 等待 ⏱️ | 起飛 → 懸停 → 降落 |
| 第5課 | 如果...就... ⚠️ | 遇障礙自動停止 |
| 第6課 | 我的飛行計劃 ✈️ | 設計並執行自己的路線 |

## 🎮 虛擬挑戰（2堂）

| 堂數 | 內容 |
|------|------|
| 第7課 | 穿越雲層 ☁️ | 在虛擬環境中避開障礙物 |
| 第8課 | 緊急救援任務 🚨 | 完成救援任務 |

---

## 🎯 學習目標

1. **序列**：先後次序的概念
2. **迴圈**：重複動作不需要寫兩次
3. **條件**：遇到情況要變通
4. **邏輯思維**：解決問題的能力

*目標：1-2 堂完成*
